# PyQt5-QWidgetDemo

此项目是移植C++版的[QWidgetDemo](https://gitee.com/feiyangqingyun/QWidgetDemo), 如有需要C++版本请点链接跳转。

### 开始

```bash
git clone https://github.com/weijiang1994/PyQt5-QWidgetDemo.git
cd PyQt5-QWidgetDemo
python3 -m venv venv
source venv/bin/active
```
